Визначення тегів

h1–h6 - відповідають за структуру сторінки і ранжуються за пріоритетом важливості.

strong - призначений для акцентування тексту. Браузери відображають такий текст жирним шрифтом.

em - призначений для акцентування тексту. Браузери відображають такий текст курсивним шрифтом.

smal - зменшує розмір шрифту на одиницю в порівнянні зі звичайним текстом.

ins - визначає текст, котрий був доданий в нову версію документу.

del - використовується для виділення тексту, який був вилучений у новій версії документа.

sub - визначає текст підрядкового тексту.

sup - відображає шрифт у вигляді верхнього індексу.

code - призначений для відображення однієї або декількох рядків тексту, який являє собою програмний код.

kbd - використовується для позначення тексту, що набирається на клавіатурі або для назви клавіш.

samp - використовується для відображення тексту, який є результатом виведення комп'ютерної програми або скрипта.

pre - визначає блок попередньо форматованого тексту.

dfn - використовується для вказівки терміну, обумовленого в контексті фрази або пропозиції.

abbr - вказує, що послідовність символів є абревіатурою..

q - використовується для виділення в тексті цитат.

cite - позначає текст як цитату або виноску на інший матеріал.

blockquote - призначений для виділення довгих цитат всередині документа.

br - вказує місце розриву рядків в тексті HTML сторінки.

hr - малює горизонтальну лінію, яка за своїм виглядом залежить від використовуваних параметрів, а також браузера.

ul - встановлює маркований список.

ol - встановлює нумерований список.

li - визначає окремий елемент списку.

a href=“” - задає адресу документа, на який слід перейти.

img - призначений для відображення на веб-сторінці зображень в графічному форматі GIF, JPEG або PNG.
