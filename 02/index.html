Визначення тегів

Теги < h1 > < /h1 > – < h6 > < /h6 > -

відповідають за

структуру сторінки

і ранжуються

за пріоритетом

важливості.
Тег < strong > - призначений для акцентування тексту. Браузери відображають такий текст жирним шрифтом.
Тег < em > - призначений для акцентування тексту. Браузери відображають такий текст курсивним шрифтом.
Тег < small > - зменшує розмір тексту.
Тег < ins > - підкреслює текст.
Тег < del > - використовується для закреслення тексту.
Тег < sub > - відображає текст у вигляді нижнього індексу.
Тег < sup > - відображає текст у вигляді верхнього індексу.
Тег < code > - призначений для відображення однієї або декількох рядків тексту, який являє собою програмний код.
Тег < kbd > - використовується для позначення тексту, що набирається на клавіатурі або для назви клавіш.
Тег < samp > - використовується для відображення тексту, який є результатом виведення комп'ютерної програми або скрипта.
Тег < pre > - визначає блок попередньо форматованого тексту.
Тег < dfn > - використовується для вказівки терміну, обумовленого в контексті фрази або пропозиції.
Тег < abbr > - вказує, що послідовність символів є абревіатурою.
Тег < q > - використовується для виділення в тексті цитат.
Тег < cite > - позначає текст як цитату або виноску на інший матеріал.
Тег < blockquote > - призначений для виділення довгих цитат всередині документа.
Тег < br > - вказує місце розриву рядків в тексті HTML сторінки.
Тег < hr > - малює горизонтальну лінію, яка за своїм виглядом залежить від використовуваних параметрів, а також браузера.
Тег < ul > - встановлює маркований список.
Тег < ol > - встановлює нумерований список.
Тег < li > - визначає окремий елемент списку.
Тег < a href=“” > - задає адресу документа, на який слід перейти.
Тег < img > - призначений для відображення на веб-сторінці зображень в графічному форматі GIF, JPEG або PNG.
